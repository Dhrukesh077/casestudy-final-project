# ✈️ SkyRoutes Airline Profitability Dashboard

## 📌 Project Overview

The **SkyRoutes Airline Profitability Dashboard** is a business-oriented data analytics project developed using **SQL** and **Microsoft Excel Dashboarding** techniques. The project analyzes airline route performance using a dataset containing 5,000 flight records.

The primary objective of this project is to identify profitable airline routes, analyze operational efficiency, evaluate occupancy performance, and generate meaningful business insights through data visualization and SQL analysis.

---

# 🎯 Project Objectives

- Identify the top 10 most frequent airline routes.
- Calculate average revenue, operational cost, and profit per route.
- Detect underperforming routes with negative profitability.
- Analyze seat occupancy percentage across routes.
- Examine monthly profit trends.
- Compare profitability between domestic and international routes.
- Rank routes based on revenue generated per minute of flight duration.
- Create an interactive Excel dashboard for visualization.

# 📊 SQL Analysis Performed

## 1. Top 10 Most Frequent Routes
Identified routes with the highest number of flights.

## 2. Revenue, Cost & Profit Analysis
Calculated average revenue, operational cost, and profit for each route.

## 3. Underperforming Routes
Detected routes where average profitability was negative.

## 4. Seat Occupancy Analysis
Calculated occupancy percentage using seats sold and available seats.

## 5. Monthly Profit Trend
Analyzed monthly fluctuations in airline profitability.

## 6. Domestic vs International Profitability
Compared profitability performance between domestic and international routes.

## 7. Revenue Per Minute Ranking
Ranked routes based on revenue generated relative to flight duration.
